Binary Search Tree

Author:
Peter Bauer

Introduction

You are to implement a BST which holds integer numbers and some functions to analyze it.

Functions to Implement

  • new_bst() Initializes the BST
  • get_depth() Returns the depth of the BST
  • add() Adds an element
  • root_value() Returns the value of the root
  • left_subtree() returns the left subtree
  • right_subtree() returns the right subtree
  • traverse_pre_order() Traverses pre order
  • traverse_in_order() Traverses in order
  • traverse_post_order() Traverses post order
  • are_equal() Checks if two trees are equal
  • most_left_longest_branch() Yields the longest branch in the tree which is at the most left position
  • get_number_of_subtrees() Returns the number of non-empty subtrees in a tree

Assignment

  • Implement the functions required to pass all unit tests
